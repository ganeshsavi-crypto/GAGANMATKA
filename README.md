# Professional Login/Signup System with Dashboard

## Features
- **Email/Password Authentication** with Firebase
- **Google OAuth Integration**
- **Password Reset** functionality
- **Responsive Design** for all devices
- **Professional Dashboard** with real-time stats
- **Form Validation** and error handling
- **Loading States** and toast notifications
- **Secure Session Management**

## Setup Instructions

### 1. Firebase Configuration
1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Create a new project
3. Enable Authentication:

   - Enable Email/Password
   - Enable Google authentication
4. Add authorized domains in Authentication settings

### 2. Project Setup
1. Clone or download the project files
2. Update Firebase configuration in `auth.js` with your project's config
3. Deploy files to a web server or use locally
